# 💳 Credit Card Fraud Detection System

## 📌 Project Overview

This project is a machine learning-based Credit Card Fraud Detection System developed using Python, Scikit-learn, and Streamlit.

The system analyzes transaction data and predicts whether a transaction is fraudulent or legitimate. It includes data preprocessing, model training, evaluation, and an interactive Streamlit dashboard.

---

## 🚀 Features

- Upload transaction CSV files
- Detect fraudulent transactions
- Data preprocessing and scaling
- Machine Learning prediction
- Fraud probability analysis
- Download prediction results
- Interactive Streamlit dashboard

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit
- Git & GitHub

---

## 📊 Dataset

Dataset used:

Credit Card Fraud Detection Dataset

- 284,807 transactions
- 30 features
- Binary classification:
    - 0 = Legitimate Transaction
    - 1 = Fraudulent Transaction

---

## 📈 Machine Learning Workflow

1. Data Cleaning
2. Feature Scaling
3. Train-Test Split
4. Logistic Regression
5. Random Forest Classifier
6. Model Evaluation
7. Streamlit Deployment
